# QA Test Project: Hundt Application (Web & Mobile)

This repository contains a comprehensive Quality Assurance project for the **Hundt** application, covering both its web and mobile platforms. It demonstrates a full-cycle testing approach, from initial planning and analysis to final reporting.

---

## üéØ Project Goal

The purpose of this project is to showcase a structured and professional approach to software testing. It includes analyzing requirements, creating a strategic test plan, designing and documenting ~100 test cases, and ensuring full requirements traceability (RTM).

---

## üõ†Ô∏è Tools & Methodologies

This project was created using a professional QA stack to simulate a real-world environment:

* **Documentation & Planning:** Confluence (exported to PDF)
* **Test Case Management:** MS Excel / Google Sheets
* **Test Design Techniques:** Application Decomposition (Mind Mapping), Requirements Analysis, Permission Testing.
* **Tracking (Simulated):** Jira / Trello
